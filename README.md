# test_AVKit

这里主要是测试相关的AVKit 下面的两个ios上的类的使用；
ios 9 之后增加了一个pip（picture in picture ）的视频播放功能(在wkWebKit有)，这里显示测试AVPlayerViewController类，类似ios 9 之前的MPMoviePlayerViewController，内容不可以修改；

问题：
